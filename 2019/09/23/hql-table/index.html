HiveQL的Table常用操作

Sep 23, 2019 | Data Warehouse | 阅读 | 411 字 | 2 分钟

Apache Hive数据仓库软件有助于读取，写入和管理驻留在分布式存储中并使用SQL语法查询的大型数据集。而Table是Hive组织数据存储的主要数据单元，是一种结构化存储，用二维表结构来表示。

0x00 create table

经常使用的表有：内部表(managed table)，外部表(external table)，分区表(partitioned table)，临时表(temporary table)等。

0x01 alter table

0x02 describe table & show table

0x03 truncate table & drop table
